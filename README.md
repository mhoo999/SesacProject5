# SesacProject5
Escape from SeSac

Duration   : 24/02/01 ~ 24/03/08
Team size  : 2
Skill      : UE5.3, BP, C++

멀티플레이 익스트랙션 루트 슈팅 장르 게임 프로젝트로 다중 인원이 특정 맵에 진입하여 타겟 제거, 아이템 획득 등의 목적을 수행하고 무사히 탈출하는 것이 목적인 게임. 멀티플레이와 사용자 조작 및 여정, AI의 역할 수행을 개발 중점으로 목표하고 진행한 프로젝트.

AI와 퀘스트, 전반적인 기획을 담당.
